源启云联 Yuanqi Cloud Link  

📖 项目简介  

源启云联是一套完整的全栈管理系统，涵盖后端服务、PC端管理后台、多端适配移动应用，适用知识付费等类业务场景，整体架构轻量化、易部署、易扩展，兼顾性能与开发效率。  

🛠️ 全栈技术架构  

后端服务   
    1. 核心架构：go,gin,gorm,redis,mysql  
    2. 小程序上传（可选）：nodejs  
    3. 文档导入（可选）：libreoffice，mupdf  
后台管理系统（PC端）  
    1. 架构：Vue3,Element Plus  
移动端应用    
    1. 框架：UniApp（一套代码多端发布）  
    2. 适配：微信小程序
