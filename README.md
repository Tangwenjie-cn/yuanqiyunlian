源启云联 Yuanqi Cloud Link

📖 项目简介

源启云联是一套完整的全栈管理系统，涵盖后端服务、PC端管理后台、多端适配移动应用，适用于企业数字化管理、云服务管控、数据运维等各类业务场景，整体架构轻量化、易部署、易扩展，兼顾性能与开发效率。

🛠️ 全栈技术架构

后端服务
    1. 核心框架：Gin（Go语言Web框架）
    2. 数据库：MySQL（持久化数据存储）
    3. 缓存中间件：Redis（热点数据缓存、会话存储）
    4. 其他：接口规范化、权限校验、日志管理
后台管理系统（PC端）
    1. 框架：Vue3 + Composition API
    2. UI组件库：Element Plus
    3. 工程化：Vite构建、路由管控、状态管理
移动端应用
    1. 框架：UniApp（一套代码多端发布）
    2. 适配：微信小程序、App、H5等多端兼容
    3. 能力：接口对接、数据展示、业务操作
